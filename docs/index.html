MacBroom

Free, open-source Mac cleaner. Built for the community.

Download View on GitHub

🧹 Smart Scan

One click sweeps caches and developer junk in parallel. Shows GB you can reclaim.

🗑️ App Uninstaller

Removes apps and their leftover files in ~/Library. Quits running apps automatically.

📦 Duplicate finder

SHA-256 hashes catch identical files anywhere. Keep newest, trash the rest.

🔍 Disk Explorer

Treemap of any folder. Click to drill in. See what's actually eating your disk.

🧠 Memory cleaner

Live RAM breakdown. One-click purge with admin auth.

🛠️ Maintenance

Flush DNS, rebuild Spotlight, repair fonts, force-empty Trash. Everything OnyX does.

🟢 Hacker Mode

Matrix rain, terminal UI, ASCII art. Toggle a global green-on-black monospace theme.

⌨️ Cmd+K palette

Raycast-style fuzzy launcher for every action in the app.
